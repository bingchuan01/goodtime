# 图标转换说明

## 重要提示

小程序tabBar**必须使用PNG格式**的图标文件，其他图标可以使用SVG格式。

## 需要转换的图标（tabBar）

以下图标需要从SVG转换为PNG格式（81x81px）：

### images/tabbar/ 目录
- `home.svg` → `home.png` (未选中状态，灰色 #999999)
- `home-active.svg` → `home-active.png` (选中状态，橙色 #FF6B35)
- `handshake.svg` → `handshake.png` (未选中状态)
- `handshake-active.svg` → `handshake-active.png` (选中状态)
- `publish.svg` → `publish.png` (未选中状态)
- `publish-active.svg` → `publish-active.png` (选中状态)
- `member.svg` → `member.png` (未选中状态)
- `member-active.svg` → `member-active.png` (选中状态)
- `user.svg` → `user.png` (未选中状态)
- `user-active.svg` → `user-active.png` (选中状态)

## 转换方法

### 方法1：使用在线工具（推荐）
1. 访问 https://convertio.co/zh/svg-png/ 或 https://cloudconvert.com/svg-to-png
2. 上传SVG文件
3. 设置输出尺寸为 81x81 像素
4. 下载PNG文件并替换到对应目录

### 方法2：使用Python脚本
如果已安装Python和cairosvg库：
```bash
pip install cairosvg
python convert_icons.py
```

### 方法3：使用ImageMagick
```bash
magick convert -resize 81x81 images/tabbar/home.svg images/tabbar/home.png
```

### 方法4：使用设计工具
- 使用Photoshop、Figma、Sketch等设计工具打开SVG文件
- 导出为PNG格式，尺寸设置为81x81px

## 其他图标（已支持SVG）

以下图标已使用SVG格式，小程序可以直接使用，无需转换：
- `images/icons/` 目录下的所有图标
- `images/logo.svg`

## 图标颜色规范

- **未选中状态**: #999999 (灰色)
- **选中状态**: #FF6B35 (橙色，主色调)
