## 🔧 如果遇到问题

### 图标不显示
- 检查图标文件路径是否正确
- 确认PNG图标文件存在于 `images/tabbar/` 目录

### 组件不显示
- 检查 `pages/index/index.json` 中的组件引用
- 确认组件文件存在于 `components/` 目录

### 编译错误
- 检查语法错误
- 查看控制台错误信息
- 确认所有文件路径正确

## 📬 站内信联调（列表 / 详情 / 写站内信 / 删除）

1. **启动站内信后端**
   - 进入 `server` 目录：`cd server`
   - 安装依赖：`npm install`
   - 启动：`npm start`，默认 `http://localhost:3000`，接口前缀 `/api/messages`

2. **小程序端配置**
   - 将 `utils/api.js` 中的 `BASE_URL` 改为 `http://localhost:3000/api`（仅开发工具访问本机时有效）
   - 登录后确保 `userInfo` 含 `id`（如 `user_1`），或本地设置 `wx.setStorageSync('userInfo', { id: 'user_1' })`、`wx.setStorageSync('token', 'user_1')`，以便请求带 `X-User-Id` / Bearer

3. **接口说明**
   - 详见 `server/README.md`
