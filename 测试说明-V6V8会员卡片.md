## 📊 模拟数据说明

当前模拟数据生成逻辑（`pages/index/index.js`）：
```javascript
coverType: index % 3 === 0 ? 'video' : 'image'
memberLevel: index % 5 === 0 ? 'V6' : (index % 7 === 0 ? 'V8' : '')
```

这意味着：
- **索引0, 3, 6, 9, 12, 15, 18, 21...** → 视频封面
- **索引0, 5, 10, 15, 20...** → V6会员
- **索引7, 14, 21, 28...** → V8会员

**重点测试项目：**
- 索引0：V6 + 视频（应不显示播放图标和时长）
- 索引21：V8 + 视频（应显示播放图标和时长，图标居中）

## 🐛 如果发现问题

### 播放图标仍然显示在V6会员卡片上
- 检查 `components/project-card/project-card.wxml` 第12行
- 确认条件判断：`wx:if="{{project.memberLevel !== 'V6'}}"`

### 时长提示仍然显示在V6会员卡片上
- 检查 `components/project-card/project-card.wxml` 第13行
- 确认条件判断：`wx:if="{{project.memberLevel !== 'V6'}}"`

### V8会员播放图标不居中
- 检查 `components/project-card/project-card.wxss` 中的 `.play-icon` 样式
- 确认使用了：`top: 50%; left: 50%; transform: translate(-50%, -50%);`
